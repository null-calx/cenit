* cenit

Simple app for viewing and manuipulating relational databases.

** setup

Describe your database in the file `database/db-structure.sexp` and run `make` in the same `database` directory.

** test

#+begin_src sh
  npm install
  npm run dev -- --open
#+end_src

** deploy

#+begin_src sh
  npm build
#+end_src
